Данное веб-приложение состоит из следующих компонентов:

1. Сервер с самим приложением на React.js. Лежит в папке 
node. Реализован на node.js, для установки необходимо 
запустить команду npm install. Для запуска сервера используется
команда npm start. Приложение получает данные с сервера базы
данных и отправляет запросы на изменение списка мероприятий.

2. Сервер доступа к базе данных. Лежит в папке falcon.
Реализован через falcon на языке python, для собственно запуска
сервера используется waitress. Для запуска сервера применяется
команда python waitress_launcher.py.

3. База данных SQL, используется PostgreSQL. Данные хранятся в
базе данных test. В этой базе данных должна быть создана следующая
таблица с названием event_list:

id			-	SERIAL, primary key
event_name	-	text[] NOT NULL
event_time	-	integer NOT NULL

Данная таблица будет содержать список мероприятий, который
обновляется при работе сайта.
